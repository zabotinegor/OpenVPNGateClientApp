# Reference Materials

This directory stores validated notes, device baselines, and supporting documentation that help explain or contextualize test runs.

## Archived Materials

### archived/
- Contains historical test validation notes, device-specific findings, and reference configuration snapshots.
- Example: `TV-PAUSE-RESUME-SMOOTHER-STATES-2026-04-25.md` — TV pause/resume transition timing notes from April 25, 2026 validation.

## Usage

When documenting test cases or suites, link to relevant reference materials:

```markdown
Reference: `tests/manual-e2e/reference/archived/TV-PAUSE-RESUME-SMOOTHER-STATES-2026-04-25.md`
```

Keep reference materials organized by topic or date to make them easy to locate and maintain.
